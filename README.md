# Functions

Criar uma função para soma de listas e substituição de um número em uma lista

## Soma

Soma todos os algarismo de uma lista

Ex: (' 1 2 3 4)  
Resultado: 10

## Substitui

Substitui um algarismo por outro em uma lista

Ex: (' 1 2 3 4) | Substituir 3 por 2  
Resultado: (' 1 2 2 4)
